STEP 1: LOAD EXTENSION IN CHROME (2 minutes)
====================================================================

1. Open Chrome
2. Type in address bar: chrome://extensions/
3. Click toggle for "Developer mode" (top right)
4. Click "Load unpacked" button
5. Browse to: C:\Users\ltken\OneDrive\Documents\GitHub\PriceCheck-Build\extension\dist
6. Click "Select Folder"
7. You should see PriceCheck card with red icon

✓ If you see the PriceCheck card, extension is loaded!

====================================================================

Double-click this file to open in Chrome:
C:\Users\ltken\OneDrive\Documents\GitHub\PriceCheck-Build\test-page.html

STEP 3: TEST THE EXTENSION (1 minute)
====================================================================

1. Press F12 to open DevTools (this is important!)
2. Click "Console" tab in DevTools
3. Look for message: "PriceCheck content script loaded"
   (If you don't see this, refresh the page)

4. Click the PriceCheck icon in Chrome toolbar (red square)
5. A popup should open

6. Click the red "Analyze This Page" button

7. Watch what happens:
   - Button says "Analyzing..."
   - Console shows logs
   - Results appear in popup

====================================================================
TROUBLESHOOTING
====================================================================

PROBLEM: Content script not loaded
→ Refresh the test page (F5)
→ Check chrome://extensions/ and click refresh on PriceCheck

PROBLEM: Popup shows error
→ Check console for error details
→ Make sure you're on the test-page.html tab

PROBLEM: No results, score is 0
→ Backend might need Claude API key
→ This is OK for testing the UI flow
→ The connection is working if you see the result box
